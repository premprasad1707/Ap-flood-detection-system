# Ap-flood-detection-system
AI-powered Flood Early Warning System for agricultural fields along canals and embankments in Andhra Pradesh using Machine Learning, Streamlit, and FastAPI.

# Run it with:-
.\venv\Scripts\activate
streamlit run app.py
